Dashboard hero animation (optional)
===================================
Drop an animation file here to show it on the user dashboard above "Congratulations on getting hired!".

Supported names (first one found is used):
  - hero.gif       (animated GIF)
  - animation.gif
  - hero.mp4       (looping video, no sound)
  - hero.webm
  - animation.mp4

Use a short, looping clip or GIF (e.g. confetti, welcome animation, your logo with motion).
Videos play automatically, muted and looped. Keep file size reasonable for fast loading.
